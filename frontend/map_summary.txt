folder_structure

frontend: 

src
│
├── MapApp
│   ├── components
│   │   ├── HospitalSearchForm.jsx        # Form for hospital search with filters
│   │   ├── HospitalCard.jsx             # Displays hospital details like name, address
│   │   ├── MapComponent.jsx             # Renders the map and updates with markers
│   │   ├── LoadingState.jsx             # Shows loading spinner or state
│   │   └── ErrorHandling.jsx            # Handles errors (e.g., no hospitals found)
│   │
│   ├── pages
│   │   ├── HospitalSearchPage.jsx       # Page for searching hospitals, integrates the form and map
│   │   └── MapPage.jsx                  # The page showing map with hospitals and routes
│   │
│   ├── store
│   │   ├── mapStore.jsx                 # Stores state for hospitals, filters, etc.
│   │
│   ├── utils
│   │   ├── api.js                       # Handles API requests to the backend
│   │   ├── mapUtils.js                  # Utility functions for map integration
│   │
│   └── MapApp.jsx                       # Main Map App page that includes routing to other pages
│
├── LandingPage
│   ├── LandingPage.jsx
│   └── App.jsx


Backend:

src
│
├── Map_Services
│   ├── controller

│   │
│   ├── routes
│   │   ├── map.routes.js            # API routes for hospital search
│   │
│   ├── services
│   │   ├── hospitalSearchService.js # Handles interaction with Overpass API
│   │
│   ├── utils
│   │   ├── overpassAPI.js           # Makes requests to Overpass API
│   │   └── mapUtils.js              # Helper functions for processing map data
│
├── index.js
